Detailed Description

The abstract AbstractWormsSimUIStrategy class (similar to a Java or C# Interface or an Objective-C Protocol) declares the functions needed by any class that displays WormsSim state to a user and/or accepts user input.

AbstractWormsSimUIStrategy collaborates as part of an implementation of the Strategy Design Pattern: https://en.wikipedia.org/wiki/Strategy_pattern The assumption is that many different concrete strategies for displaying WormsSim state to a users and accepting user input may exist. For example, there may be a terminal (Curses) based textual display or a 2D vector graphics display or a 3D display. Similarly, user input may come from a keyboard or over a network connection or from a data file or from script.
