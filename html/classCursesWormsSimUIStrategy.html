Detailed Description

Instances of CursesWormsSimUIStrategy provide NCurses based graphical display and user interaction of WormsSim instances.

Design Notes:

  • CursesWormsSimUIStrategy implements the interface specified by the abstract AbstractWormsSimUIStrategy class and participates in the Strategy design pattern to decouple display and user input from simulation encapsulation.
  • CursesWormsSimUIStrategy adheres to the practical advice from Scott Meyers, “More Effective C++: 35 New Ways to Improve Your Programs and Designs 1st Edition” Item 33, “Make non-leaf classes abstract” as elaborated to use the Template Method design pattern in “Virtuality: C/C++ Users Journal, 19, September 2001” http://www.gotw.ca/publications/mill18.htm
